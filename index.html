Daily Workflow

This section provides a step-by-step guide to the daily ad trafficking process. Click each step header to expand and see the detailed procedure.

The first task of the day is to check the Canoe platform for reports on missing ad promotions. The email arrives twice daily and contains two attachments (one for paid, one for unpaid promos).

Detailed Steps:

  1. Open the "Promo Missing Ads" email from Canoe.
  2. To analyze the attached report, first sort by **Network (Column B)**.
  3. Apply a highlight marker to **ABC** and **Disney Channel** rows. This is critical because **Charter** only receives ads for these two networks. Any other network flagged as missing for Charter can be ignored.
  4. Next, sort the report by affiliate (**Comcast - Column N** and **Cox - Column P**) to group them.
  5. Compare the **Mass ID** on the report to your pre-existing tracking list to identify new issues.
  6. If an item is new and valid, proceed to the action flow below.
  7. If an item was already repitched but remains on the report for a few days, escalate by emailing Canoe Operations.

Action Flow for New Missing Ads

Is the item a "MOVE" asset (e.g., FX, starts with `MOVE...`)?

YES → Email Comcast

Cannot be repitched by our team. Email Christopher Freeman to request a repitch.

NO → Standard Repitch

Search the Mass ID in Media Hub (VOD+) and repitch to the correct affiliate.

This workflow uses the "FX & Nat Geo Working" Google Doc and requires converting standard assets into DAI (Dynamic Ad Insertion) assets.

Detailed Steps:

  1. **Clean the Google Doc (bi-weekly):** Review the "License Window End" column. Cut and paste any expired ads into the "Pitched - Past LW End" tab.
  2. **Check for "Awaiting Content":** Find rows marked with this status. Copy the **ISCI/Provider ID (Column D)**.
  3. **Search in Media Dashboard:** Paste the ID into the search bar in the **Media Dashboard** (not VOD+). If the status is green/completed, proceed. If not, the content is not yet available.
  4. **Generate DAI Asset:** In the Media Dashboard, check the box for the completed asset and from the dropdown menu, select "Generate DAI asset".
  5. **Create Planner in VOD+:** Once the DAI asset is generated, switch to **VOD+**. Filter by **NLDD Type: DAI** and **Start Date: Today**. The new asset will appear.
  6. Select the new asset(s) and click **"Create Planner"**.
  7. **Format and Update Google Doc:** From the planner, narrow down the columns to **Asset Name, Move Asset ID, Start,** and **End**. Copy the HD and SD lines and paste them into the FX Google Doc. Update the status column to "Complete".

This is a multi-team workflow tracked within the "ABC & Kids Working" Google Doc. Our work begins after Media Services has completed their prep.

Detailed Steps:

  1. **Monitor "Media Service" Column (N):** The workflow begins when an **"X"** appears in this column, indicating Media Services has prepped the ad.
  2. **Verify in VOD+:** Copy the **Placing ID (Column O or Q)** and search for it in VOD+.
  3. Confirm that the correct versions exist (one SD and one HD) and that the media is correct.
  4. If the Placing IDs are incorrect (e.g., HD in SD slot), highlight the row and notify the Ad Manager. Do not change the IDs.
  5. **Update Google Doc:** If the assets are correct, copy the **Mass ID** from VOD+.
  6. Paste the Mass ID into the appropriate SD/HD column in the Google Doc and add the current date to the last column.

Troubleshooting Guide

Encounter a problem? This section provides a quick guide to common issues and their resolutions. Select an issue from the list below to see the step-by-step fix.

Select an issue to see details.

Resolving GAM Issues

  1. **Initial Repitch:** Repitch the asset to the affiliate in VOD+. Ask the Ad Manager to check again in 30 minutes.
  2. **Check with Canoe:** If the issue persists, email Canoe Operations to confirm they see the asset as delivered.
  3. **Escalate:** If widespread, inform Caroline's team via the `Canoe STB VOD troubleshooting` Teams chat. Contacts: Caroline Bilodeau, Jessica Skold, Torey Beigh. They will open the official ticket.

Resolving Missing Media

  1. **Check Edit History:** Right-click the cell in the Google Doc to see *when* the "X" was added. If it was very recent, wait 30-60 minutes.
  2. **Contact Media Services:** If still missing, ping Sean Smith via Teams or email the group at `datg.media.services@disney.com` to confirm the asset is in a "ready state."
  3. **Purge and Republish:** If Sean confirms it's ready, the asset may be stuck. **Purge it from VOD+ and SDVI**, then find the asset in **Banjo** and click **"Publish"** to resend it.

Handling Duplicate Ads

  1. **Mark as Duplicate:** Highlight the entire row with the designated "duplicate" color and put an "X" in the processing columns.
  2. **Add a Note:** Add a note indicating which line it is a duplicate of (e.g., "Duplicate of line 56").
  3. **Await Instructions:** Contact the **Ad Manager and their Supervisor** (contact info in Google Doc) to clarify if this is a true duplicate or an updated version.
